How the Internet Actually Works

1. You type a URL

It starts with a request. You type google.com. Your computer doesn't know where that is yet.

2. The DNS Lookup

Your computer asks the "Phonebook of the Internet" (DNS).

"Where is google.com?"

3. The Handshake

The server replies. Your browser starts talking. If this line isn't encrypted (HTTP), anyone can listen in.

4. The Attack

Oh no! A "Man in the Middle" intercepts your request. This often happens on insecure public Wi-Fi.

5. Fake Websites (Phishing)

Sometimes the server you reach isn’t the real one. It only looks right. A single wrong letter can send your password to an attacker.

Example: g00gle.com vs google.com

6. Weak Passwords

If you reuse passwords, one leak unlocks everything. Hackers don't guess passwords anymore; they reuse stolen ones.

The Defense

7. HTTPS (The Shield)

HTTPS encrypts the conversation. Even if a hacker intercepts it, they only see scrambled nonsense.

Note: Encryption hides what you say, not who you talk to.

5 Things That Actually Keep You Safe

  • ✔ Use a Password Manager

    It creates strong, unique passwords so you don’t have to remember them.

  • ✔ Enable 2FA

    Two-Factor Authentication means even if they have your password, they can't get in.

  • ✔ Check the URL

    Designs can be faked. The URL bar (google.com) cannot lie.

  • ✔ Avoid Public Wi-Fi

    If you must use it, don't log into banks. Or use a VPN.

  • ✔ Don’t Panic

    Security is about habits, not fear.
